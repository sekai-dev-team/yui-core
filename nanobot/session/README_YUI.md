# Yui 核心引擎：会话管理 (Session) - 深度解析

本模块负责 Yui 聊天历史的持久化、缓存及上下文窗口的智能截断。

## 核心组件

### 1. `manager.py` (会话管理器)
- **JSONL 存储**: 每一轮对话都实时追加到 `.jsonl` 文件中。这种流式格式确保了数据的极端稳定性。
- **状态追踪**: 记录会话的 `last_consolidated` 游标，配合 `memory` 模块实现定时记忆整理。
- **智能截断 (`get_history`)**: 能够自动将原始会话记录脱水（剔除冗余的时间戳和内部元数据），只保留 LLM 真正关心的内容。

## 开发者备忘 (Yui 魔改点)
- **多端同步基础**: 通过 `channel:chat_id` 的 Key 映射，我们可以在此层级增加“跨平台账号关联”逻辑。
- **会话搜索**: 后续若要实现“搜索以往聊天记录”的功能，可在此模块引入简单的全文索引。
- **冷热分离**: 对于极长会话，可以修改 `_load` 逻辑，只加载最近几轮，其余内容通过 RAG 动态拉取。
