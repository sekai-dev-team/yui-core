# Yui 核心引擎：工具函数 (Utils) - 深度解析

本模块提供系统级的底层支撑，主要负责路径管理、字符串处理和文件名安全化。

## 核心功能

### 1. 路径管理 (`helpers.py`)
- **`ensure_dir`**: 递归创建目录，确保 IO 操作前路径可用。
- **`get_data_path`**: 统一定义全局数据根目录 (`~/.nanobot`)。
- **`get_workspace_path`**: 定义 Agent 的主要活动区，这是所有技能、记忆和临时文件的存放地。

### 2. 安全与转换
- **`safe_filename`**: 将包含非法字符（如 `:`, `*`, `?` 等）的会话 ID 转换为安全的文件名，防止在 Windows 等平台上出现持久化错误。
- **`parse_session_key`**: 将 `channel:chat_id` 格式的字符串解析回结构化数据。

### 3. 内容处理
- **`truncate_string`**: 用于在日志回显或简报中截断过长的输出。
- **`timestamp`**: 提供统一的 ISO 格式时间戳。

## 开发者备忘 (Yui 扩展点)
- **多工作区支持**: 若要运行多个独立的 Yui 实例，可以通过 `get_workspace_path` 的参数进行动态隔离。
- **扩展工具集**: 后续若有通用的图片处理、PDF 解析等轻量级逻辑，可考虑在此增加辅助函数。
