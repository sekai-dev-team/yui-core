# Yui 核心引擎：心跳自省 (Heartbeat) - 深度解析

本模块实现了 Yui 的“主动意识”触发机制，让 Agent 能够周期性地检查并处理背景任务。

## 核心组件

### 1. `service.py` (心跳引擎)
- **触发逻辑**: 默认每 30 分钟（可配置）执行一次。
- **文件驱动**: 它的唯一指令来源是 `workspace/HEARTBEAT.md`。
- **自省循环**: 
    1. 扫描 `HEARTBEAT.md`。
    2. 若发现非空（且不只是标题或空复选框），则向 Agent 发送强力提醒词。
    3. 等待 Agent 处理任务并返回 `HEARTBEAT_OK`。

## 开发者备忘 (Yui 魔改点)
- **主动权扩展**: 目前仅扫描文件。我们可以修改 `_tick` 逻辑，增加对数据库、RSS 更新或 API 状态的扫描，主动告知 Agent “出事了”。
- **频率动态化**: 可以根据任务紧急程度，让 Agent 自动修改心跳间隔。
